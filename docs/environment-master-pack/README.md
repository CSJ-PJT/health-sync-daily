# Longest Dawn Environment Master Pack

Style target: A+B

- A: MapleStory2-like readability, clean silhouettes, friendly stylized forms
- B: Rust / State of Decay-like grounded survival mood, muted material realism
- Final target: grounded stylized 3D environment, readable from quarter/isometric camera, not photorealistic, not overly cute, not noisy

## Purpose
Use this pack as the source of truth for the next environment quality pass.

The current scene should not be redesigned from scratch. Improve it through:

1. texture and material replacement
2. material separation
3. lighting and shadows
4. ambient occlusion / contact shadow
5. readability and zone identity

## Priority
1. Ground materials
2. Walls and building exteriors
3. Roofs
4. Props
5. Overlay decals
6. Lighting and post-process

## Required result
The current in-game screen should look less like flat colored blocks and more like a polished stylized 3D survival settlement.

Keep:
- current map layout
- current camera angle
- current UI
- current interaction markers
- current gameplay logic

Avoid:
- clutter
- full photorealism
- chibi/cartoon exaggeration in environment
- saturated fantasy colors
- tiny noisy detail that hurts mobile readability
