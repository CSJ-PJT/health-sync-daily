# Fifth Dawn Deep Stake

## Purpose

Deep Stake is a fictional science-fantasy system layer inside Fifth Dawn.
It adds:

- alignment drift
- resonance band classification
- faction affinity
- ascension-stage scaffolding
- 4D indirect-support signals
- 5D world and social-hub foundations

It does not replace the current life-sim loop.
It sits on top of farming, village recovery, settlement growth, and quest progression.

## Current visible slice

- Resonance band card in the right-side guide panel
- alignment metrics
- initial narrative choices
- faction affinity readout
- sensed support/omen panel
- 5D gate/world hint panel

## Current gameplay hooks

- quest completion nudges Deep Stake state
- settlement object placement nudges Deep Stake state
- explicit player decisions can push luminous or shadow alignment

## Persistence

Deep Stake currently persists inside the existing life-sim save blob.
No raw health data is stored or exposed here.

## Future expansion

- grouped resonance cohorts
- faction pressure / war layers
- 4D indirect support actions
- 5D residences and social hubs
- post-ascension return / stewardship loops
