# Codex Start Prompt

Use the following prompt when starting Codex work for this reference.

```text
Read `docs/CODEX_CHARACTER_REFERENCE.md` first.

Then implement the requested UI, avatar, character, placeholder art, or visual component so it matches that reference.

Requirements:
- cute but grounded stylized 3D quarter-view feeling
- mobile-readable silhouette first
- slightly larger head only for readability
- natural human proportions
- practical modern workwear
- muted colors
- restrained facial expression
- low-poly-inspired presentation
- not comedic cartoon
- not chibi / not super-deformed

Support these role archetypes when relevant:
1. player character: modern recovery-zone survivor, backpack, improvised gear
2. archivist: records keeper, notes/documents, glasses, headset
3. worker NPC: farm/recovery-zone worker, work clothes, hat, tool

Implementation rules:
- keep silhouettes readable on small screens
- avoid bright saturated cartoon styling
- prefer reusable components and clean file structure
- if final assets do not exist, build placeholder components that match the reference tone
- before editing, briefly summarize which files you will change
- after editing, summarize what was implemented and what remains
```
