# Codex Character Reference

This document transcribes the uploaded image into a Codex-readable implementation reference.

## Overall tone
- Cute but grounded stylized 3D quarter-view RPG characters
- Readable on mobile
- Slightly larger head for readability
- Separated arms and legs
- Natural human proportions with a charming silhouette
- Practical modern work clothes
- Muted colors
- Melancholic modern atmosphere
- Not cartoon comedy
- Not super-deformed
- Expressive but restrained face
- Low-poly stylized game concept art
- Readable like Project Zomboid, but softer and more lovable

## Player character
- Fictional modern recovery-zone survivor

- Restrained expression
- Low-poly style

## Archivist character
- Archive / records keeper survivor
- Archive-oriented outfit
- Holds notes and documents
- Wears glasses and a headset
- Restrained expression

## Worker NPC
- Farm or recovery-zone worker survivor
- Work clothes and a hat
- Holds a tool
- Restrained expression
- Low-poly style

## Implementation constraints for Codex
- Preserve quarter-view readability first
- Keep silhouettes clear at small sizes
- Avoid exaggerated chibi body ratios
- Avoid bright saturated cartoon colors
- Prefer soft, practical, game-ready proportions
- Use this file as the source of truth for visual tone and character structure
