# First-Screen Visual Target

Scope:
- Active project: `unity/DeepStake3D`
- Date: `2026-04-25`
- Purpose: define a clean target for the first-screen visual direction before more polish work

Target package contents:
- `target-composition.md`
- `target-checklist.md`

Core target:
- player -> path -> notice board should read immediately
- notice board should be the first clear objective landmark
- buildings should frame the objective, not bury it
- background should hide the test-map grid quietly
- ground should be calm and secondary, not the main visual feature

This package exists to stop unstructured iteration and force the next pass to aim at a fixed screenshot target instead of accumulating mixed experiments.
